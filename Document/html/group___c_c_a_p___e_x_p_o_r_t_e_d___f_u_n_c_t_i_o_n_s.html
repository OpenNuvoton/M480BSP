◆ CCAP_Open()

void CCAP_Open ( uint32_t  u32InFormat,
uint32_t  u32OutFormet 
)

Open engine clock and sensor clock.

Parameters
[in]u32InFormatThe bits corresponding VSP, HSP, PCLK, INFMT, SNRTYPE, OUTFMT, PDORD and PNFMT configurations.
  • VSP should be ether CCAP_PAR_VSP_LOW or CCAP_PAR_VSP_HIGH
  • HSP should be ether CCAP_PAR_HSP_LOW or CCAP_PAR_HSP_HIGH
  • PCLK should be ether CCAP_PAR_PCLKP_LOW or CCAP_PAR_PCLKP_HIGH
  • INFMT should be ether CCAP_PAR_INFMT_YUV422 or CCAP_PAR_INFMT_RGB565
  • SNRTYPE should be ether CCAP_PAR_SENTYPE_CCIR601 or CCAP_PAR_SENTYPE_CCIR656
  • OUTFMT should be one of the following setting
    • CCAP_PAR_OUTFMT_YUV422
    • CCAP_PAR_OUTFMT_ONLY_Y
    • CCAP_PAR_OUTFMT_RGB555
    • CCAP_PAR_OUTFMT_RGB565
  • PDORD should be one of the following setting
    • CCAP_PAR_INDATORD_YUYV
    • CCAP_PAR_INDATORD_YVYU
    • CCAP_PAR_INDATORD_UYVY
    • CCAP_PAR_INDATORD_VYUY
    • CCAP_PAR_INDATORD_RGGB
    • CCAP_PAR_INDATORD_BGGR
    • CCAP_PAR_INDATORD_GBRG
    • CCAP_PAR_INDATORD_GRBG
  • PNFMT should be one of the following setting
    • CCAP_PAR_PLNFMT_YUV422
    • CCAP_PAR_PLNFMT_YUV420
[in]u32OutFormetCapture output format, should be one of following setting
  • CCAP_CTL_PKTEN
Returns
None

Initialize the Image Capture Interface. Register a call back for driver internal using

Returns
