#define OTG_CLR_INT_FLAG (   u32Mask)

This macro is used to clear OTG related interrupt flags.

Parameters
[in]u32MaskThe combination of interrupt source. Each bit corresponds to a interrupt source. Valid values are listed below.
  • OTG_INTSTS_ROLECHGIF_Msk
  • OTG_INTSTS_VBEIF_Msk
  • OTG_INTSTS_SRPFIF_Msk
  • OTG_INTSTS_HNPFIF_Msk
  • OTG_INTSTS_GOIDLEIF_Msk
  • OTG_INTSTS_IDCHGIF_Msk
  • OTG_INTSTS_PDEVIF_Msk
  • OTG_INTSTS_HOSTIF_Msk
  • OTG_INTSTS_BVLDCHGIF_Msk
  • OTG_INTSTS_AVLDCHGIF_Msk
  • OTG_INTSTS_VBCHGIF_Msk
  • OTG_INTSTS_SECHGIF_Msk
  • OTG_INTSTS_SRPDETIF_Msk
Returns
None

This macro will clear OTG related interrupt flags specified by u32Mask parameter.
